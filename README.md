# Rased

Rased is a responsive Arabic RTL dashboard concept for monitoring cold-chain shipments. It is built with React, Vite, and Tailwind CSS, and showcases a polished logistics interface with shipment tracking, operational insights, and map-based monitoring.

## Overview

This project is a frontend prototype focused on the user experience of a shipment monitoring platform. The interface includes:

- A dashboard home view with shipment health summaries and recommendations
- A shipment management view with filtering, search, and detailed monitoring panels
- A live-style tracking map view with animated vehicle progress
- An analytics/info view for risk and status summaries
- A first-visit authentication modal and loading screen
- Mobile and desktop responsive navigation

## Tech Stack

- React 19
- Vite 8
- Tailwind CSS 4
- ESLint
